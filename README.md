# 🎵 Groovra — Premium Music Streaming App

Groovra is a modern, fully responsive music streaming web application designed to provide a seamless audio experience across desktop and mobile devices. 

## 🚀 Live Demo
**[Listen on Groovra (Live on Vercel) 🎧](https://music-app-web-fawn.vercel.app)**

## ✨ Key Features
- **Secure Authentication:** Integrated Google and Phone OTP login using Firebase Authentication.
- **Real-Time Analytics:** Tracks actual user listening time and top artists dynamically (No dummy data).
- **Mobile-First Design:** Features a custom mobile bottom navigation bar (Spotify-style) for seamless phone usability.
- **Modern Workflow:** Built utilizing advanced AI-assisted development tools like Antigravity for logic structure, Stitch for UI/UX refinement, and Ralph for code integration.

## 🛠️ Tech Stack
- **Frontend:** React, Next.js, Tailwind CSS
- **Backend/BaaS:** Firebase (Auth & Firestore)
- **Deployment:** Vercel
